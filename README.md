# 货物进销管理系统 python851

一款功能全面的货物进销管理系统，采用Python语言开发，提供完整的源代码，确保系统可运行。

## 技术栈

- 后端：Python Flask
- 前端：HTML、CSS、JavaScript
- 数据库：MySQL
- 开发工具：PyCharm
- 运行环境：Python 3.0以上、MySQL 8.0

## 功能模块

系统包含以下功能模块：

- **用户模块**
  - 登录注册
  - 个人信息修改
- **采购管理**
  - 采购订单管理
- **商品信息管理**
  - 商品信息维护
- **供应商管理**
  - 供应商信息维护
- **销售管理**
  - 销售订单管理
- **仓库管理**
  - 库存管理
- **成员管理**
  - 管理员与成员权限管理

## 系统角色

- 系统管理员：负责系统维护、成员管理及权限分配
- 普通用户：进行登录、查看和操作个人相关的进销数据

## 运行环境

- 操作系统：兼容多种操作系统
- 后端依赖库：Flask等
- 前端依赖库：无特殊要求
- 数据库：MySQL 8.0

## 核心亮点

- 基于Python Flask，易于维护和扩展
- 前后端分离，便于多终端适配
- 使用MySQL数据库，数据存储稳定可靠
- 提供完整的源代码，便于二次开发

## 部署步骤

1. 确保Python 3.0以上版本和MySQL 8.0数据库已安装
2. 克隆或下载项目源代码
3. 按照要求配置数据库连接信息（`config.py`）
4. 创建虚拟环境（可选）
5. 安装依赖库（`pip install -r requirements.txt`）
6. 运行`run.py`启动服务
